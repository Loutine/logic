\subsection{思维形式}
\label{sec:org4abe95d}
同一类型得思维形态所共有 得具有得思维形式就称作该思维的逻辑形式，又称为形式结构，简称公式
其中保持不变的是逻辑常量，变化的是逻辑变量。
\subsubsection{第一组 （命题）所有S是P}
\label{sec:org9a7504d}
\begin{itemize}
\item 所有商品都是有价值的。
\item 所有物质都是可分割的。
\item 所有发展都是由事物内部矛盾引起的。
\end{itemize}
\subsubsection{第二组 （命题）如果p,那么q}
\label{sec:org0f4d7d1}
\begin{itemize}
\item 如果为人民利益而死，就比泰山还重。
\item 如果一个人得了肺病，那么这个人就会发烧。
\item 如果出现了阶级，那么就会出现国家。
\end{itemize}
\subsubsection{第三组 （推理，因果关系）所有M都是P;所有S都是M}
\label{sec:orge6d7530}
\begin{itemize}
\item 所有学生都应该知书识礼。            所有M都是P
\item 所有大学生都是学生。                所有S都是M
\item -------------------------------
\item 所以，所以大学生都应该知书识礼。    所以，所有S都是P
\end{itemize}

\subsubsection{逻辑常项}
\label{sec:org224085b}
定义：指的是一个逻辑形式中具有确切含义，其含义始终保持不变得部分
如下：“ \textbf{如果}\ldots{}\ldots{} \textbf{那么} \ldots{}\ldots{}"
\subsubsection{逻辑变项}
\label{sec:orgc19c9aa}
定义：指一个逻辑形式中没有确切含义，可用不同词项或者命题进行替换的部分。
如下：S,P等等。
\subsubsection{词项变项}
\label{sec:org3bdfb99}
通常用 \uline{大写字母} ：S,P,M表示。
\subsubsection{命题变项}
\label{sec:org27b2dbe}
一般使用 \uline{小写字母} 表示：p，q，r，s。
